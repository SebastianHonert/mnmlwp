== Description ==

The mnmlWP theme is a free, lightweight, and easy to use WordPress theme trying to make the development of WordPress based websites simple, efficient, and fun again.

The mnmlWP theme is completely free. A link to the open source repository will be added shortly. You’re welcome!

The mnmlWP theme tries to generate little overhead. Websites build with mnmlWP are fast and efficient.

The mnmlWP theme design is 100% responsive. No matter what device, the layout will neatly adapt to the available screen sizes.

The mnmlWP theme uses the internal WP customizer and allows you to configure your theme’s look and feel according to your individual needs.

The mnmlWP theme is ready for internationalization and includes English and German language files for both frontend and backend by default.

== Installation ==

= From within WordPress =
1. Visit "Appearance > Themes > Add New"
2. Search for "mnmlWP"
3. Install and activate
4. Install recommended Plugins
5. Install child theme from [https://mnmlwp.de/download](https://mnmlwp.de/download)
6. Install shortcode plugin from [https://mnmlwp.de/download](https://mnmlwp.de/download)
7. Import demo content (optional)

== Frequently Asked Questions ==

= Is mnmlWP Free? =
Yes, mnmlWP is a free theme, and it always will be.

= Does mnmlWP have a premium version? =
Currently, no. But additional premium components are on the todo list.

= Where can I find documentation? =
The theme includes a documentation online and in the demo data packaged in the theme itself. Please find the current documentation [here](https://mnmlwp.de/documentation/).

= Do you offer support? =
We are looking forward to provide support for the free theme in the [WordPress.org forums](https://wordpress.org/support/theme/mnmlwp).

We try to answer all questions within a reasonable amount of time.

= Where can I find the theme options? =
Most of the theme's options can be found in the WP Customizer (Appearance > Customize).

= Does mnmlWP provide widget areas? =
Yes, there are 5 widget areas that you can configure in Appearance > Widgets. The theme also includes 2 additional widgets: "mnmlWP Colored Categories" and "mnmlWP Recent Posts".

= Does the theme include demo content? =
Yes, the theme comes with demo content. Simply install the WP plugin [One Click Demo Import](https://de.wordpress.org/plugins/one-click-demo-import/) and follow the instructions after the theme activation.
